# UNIT Protocol (UNIT)

UNIT Protocol is a BEP-20 utility token deployed on BNB Smart Chain, designed with transparent on-chain configuration, verifiable token mechanics, and a long-term utility-focused roadmap.

UNIT is not designed as an investment product and does not provide any guarantees of value, profit, or returns.

## Token Mechanics
UNIT Protocol implements configurable on-chain mechanics enforced directly at smart contract level:
- Burn fee (basis points, configurable)
- Treasury fee (basis points, configurable)
- Anti-whale transaction limits (configurable)

All parameters are publicly verifiable on-chain via BscScan.

---

## Liquidity & Market Structure
Initial liquidity is provided on a BNB Smart Chain decentralized exchange.

Liquidity is intended to be progressively locked using third-party liquidity locking solutions (e.g. Unicrypt / UNCX), in stages aligned with the project’s development milestones and ecosystem growth.

Liquidity management actions, where applicable, are designed to prioritize transparency and long-term protocol sustainability.
