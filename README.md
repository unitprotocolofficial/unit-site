# UNIT Protocol (UNIT)

UNIT Protocol is a BEP-20 utility token deployed on BNB Smart Chain, designed with configurable on-chain economics, transparent control mechanisms, and a long-term utility-driven roadmap.

## Official Links
- Website: https://unitprotocolofficial.github.io/unit-site/
- Whitepaper (PDF): ./UNIT_Protocol_Whitepaper.pdf

## On-chain Details
- Network: BNB Smart Chain (BEP-20)
- Contract Address: 0x4f5710c96C446b133C59Af006A366d07414B52a9
- Maximum Supply: 1,000,000,000 UNIT
- Decimals: 18

## Token Mechanics
UNIT Protocol implements configurable on-chain mechanics controlled by the contract owner:
- Burn fee (basis points, configurable)
- Treasury fee (basis points, configurable)
- Anti-whale transaction limits (configurable)

All parameters are enforced directly at smart contract level.

## Ownership & Control
Ownership is currently active to allow protocol configuration, future integrations, and ecosystem expansion.
Ownership status may be updated according to the project roadmap.

## Utility Scope
- Token-gated access and platform utilities
- Governance signaling (non-binding)
- Rewards, incentives, and future staking mechanisms
- Ecosystem integrations and partnerships

## Transparency Notice
All token mechanics, limits, and parameters are enforced on-chain and can be independently verified via BscScan.
